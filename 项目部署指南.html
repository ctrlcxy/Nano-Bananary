🍌 Nano Bananary

香蕉超市 - AI 图像与视频生成工具

📖 项目简介

Nano Bananary（香蕉超市） 是一个强大的 AI 图像与视频生成工具，基于 Google Gemini 2.5 Flash AI 模型开发。 它让你可以轻松地对图片进行各种创意编辑和生成，还能将图片转换成视频！

💡 简单来说

就像一个"魔法工具箱"，你可以：

  • 上传一张图片，让 AI 帮你换背景、修改细节
  • 让 AI 扩展图片的内容（比如把一张半身照变成全身照）
  • 将图片转换成动态视频
  • 用画笔圈出想修改的地方，AI 会智能地帮你改变
  • 一步步连续编辑，让创意无限延伸

✨ 核心功能

🎨 图像编辑

支持多种 AI 图像编辑模式，无需复杂的提示词，点击即可使用

🎬 视频生成

将任何生成的图片一键转换为动态视频，让静态画面动起来

🖌️ 局部涂选

用画笔圈出想要修改的区域，AI 会智能识别并进行编辑

🔄 连续编辑

每次输出都能作为下一次编辑的输入，创意无限衔接

🌓 主题切换

支持浅色/深色主题一键切换，护眼舒适

🌏 多语言支持

支持中文/英文界面切换，使用更便捷

🛠️ 技术栈

本项目使用以下技术构建：

React 19
TypeScript
Vite
Google Gemini AI
🤔 什么是这些技术？
  • React：一个用于构建用户界面的 JavaScript 库（类似盖房子的框架）
  • TypeScript：JavaScript 的超集，让代码更安全（就像给代码加了安全防护）
  • Vite：一个超快的开发工具（就像项目的"快速启动器"）
  • Google Gemini AI：谷歌的 AI 模型，负责处理图像和视频生成

🚀 部署步骤（详细版）

0 准备工作 - 获取 Gemini API 密钥

在开始部署之前，你需要先获取 Google Gemini 的 API 密钥（就像一把"钥匙"，让你的项目能够使用 AI 功能）。

📝 获取步骤：
  1. 访问 Google AI Studio
  2. 点击右上角的 "Get API key" 按钮
  3. 如果还没有登录，用你的 Google 账号登录
  4. 创建一个新项目（或选择已有项目）
  5. 点击 "Create API key"
  6. 复制生成的 API 密钥（类似这样：AIzaSyD...）
⚠️ 重要提示：
  • API 密钥就像你的密码，千万不要分享给别人
  • 复制后保存到安全的地方，一会儿要用到
  • 如果不小心泄露了，可以在 Google AI Studio 中删除并重新创建

1 打开终端（Terminal）

终端是一个让你通过输入命令来控制电脑的工具。

💻 如何打开终端？

在 macOS 上：

  1. 方法一：按 Command (⌘) + 空格，输入 "Terminal"，按回车
  2. 方法二：打开 启动台 → 其他 → 终端
  3. 方法三：打开 访达 → 应用程序 → 实用工具 → 终端
⚠️ 注意事项：
  • 终端窗口打开后会显示一些文字，这是正常的
  • 光标在闪烁的地方就是你输入命令的地方
  • 不要关闭终端窗口，接下来的所有步骤都在这里完成

2 进入项目文件夹

告诉电脑你要在哪个文件夹里工作。

在终端中输入以下命令：
cd /Volumes/ORICO/编程项目/Nano-Bananary
📖 命令解释：
  • cd 是 "change directory" 的缩写，意思是"进入某个文件夹"
  • 后面跟着的是项目所在的完整路径
⚠️ 常见问题：
  • 如果提示 "No such file or directory"，说明路径不对，检查项目是否在这个位置
  • 输入后按 回车键 执行命令
  • 执行成功后，终端会显示新的路径
💡 小技巧：

不想手动输入路径？可以直接把项目文件夹拖到终端窗口里，路径会自动填充！

  1. 在终端中输入 cd （cd 后面有个空格）
  2. 打开访达，找到项目文件夹
  3. 把文件夹拖到终端窗口里
  4. 按回车键

3 安装项目依赖

下载项目运行所需的所有"零件"（就像组装玩具前要先准备好所有零件）。

在终端中输入以下命令：
npm install
📖 命令解释：
  • npm 是 Node.js 的包管理工具（已经安装在你的电脑上）
  • install 意思是"安装"
  • 这个命令会自动下载并安装 package.json 文件中列出的所有依赖
⚠️ 注意事项：
  • 这个过程可能需要 1-5 分钟，取决于你的网速
  • 你会看到终端里滚动很多文字，这是正常的，表示正在下载
  • 等待出现类似 "added XXX packages" 的提示，说明安装完成
  • 如果网速慢，可能需要等待更长时间，不要中途关闭终端
🔧 遇到错误怎么办？
  • 如果提示权限错误，可以尝试运行：sudo npm install（会要求输入你的电脑密码）
  • 如果提示网络错误，检查网络连接，或者稍后再试
  • 如果有警告（WARNING），通常可以忽略，只要不是错误（ERROR）就行
✅ 安装成功的标志：

终端最后会显示类似这样的信息：

added 287 packages, and audited 288 packages in 45s

并且会在项目文件夹中生成一个 node_modules 文件夹（里面包含了所有下载的依赖）

4 配置 API 密钥

把刚才获取的 Gemini API 密钥配置到项目中，让项目能够使用 AI 功能。

📝 详细步骤：
  1. 打开代码编辑器（比如 VS Code）
  2. 打开项目文件夹：
    • 在 VS Code 中：菜单栏 → 文件 → 打开文件夹
    • 选择 /Volumes/ORICO/编程项目/Nano-Bananary
  3. 创建配置文件：
    • 在项目根目录（最外层）右键 → 新建文件
    • 文件名必须是：.env.local（注意最前面有个点）
  4. 写入 API 密钥：在新建的文件中输入以下内容
.env.local 文件内容：
GEMINI_API_KEY=你的API密钥
💡 示例：

假设你的 API 密钥是 AIzaSyD1234567890abcdefghijklmn，那么文件内容应该是：

GEMINI_API_KEY=AIzaSyD1234567890abcdefghijklmn
⚠️ 重要注意事项：
  • 等号两边 不要有空格
  • API 密钥 不要加引号
  • 文件名必须是 .env.local，不是 env.local 或其他名字
  • 保存文件后，关闭编辑器
🔍 如何确认文件创建成功？

在终端中运行以下命令：

ls -la

应该能看到 .env.local 文件在列表中

🛡️ 安全提示：
  • .env.local 文件不会被上传到 Git（代码版本控制工具）
  • 这样可以保护你的 API 密钥不被泄露
  • 绝对不要把这个文件分享给别人

5 启动项目

现在一切准备就绪，可以启动项目了！

在终端中输入以下命令：
npm run dev
📖 命令解释：
  • npm run dev 意思是"以开发模式运行项目"
  • dev 是 development（开发）的缩写
🎉 启动成功的标志：

终端会显示类似这样的信息：

VITE v6.2.0 ready in 500 ms ➜ Local: http://localhost:5173/ ➜ Network: use --host to expose ➜ press h + enter to show help

看到这个说明项目已经成功启动了！

⚠️ 注意事项：
  • 终端窗口 不要关闭，关闭后项目就停止运行了
  • 如果要停止项目，按 Control (⌃) + C
  • 端口号（5173）可能会不同，以实际显示为准

6 在浏览器中访问项目

现在可以在浏览器中查看和使用你的项目了！

🌐 访问方法：
  1. 打开你的浏览器（推荐使用 Chrome 或 Safari）
  2. 在地址栏输入：http://localhost:5173
  3. 按回车键
  4. 或者直接在终端中按住 Command (⌘) 键，然后点击链接
🎊 恭喜！

如果一切顺利，你应该能看到 Nano Bananary 的界面了！

现在你可以：

  • 上传图片进行 AI 编辑
  • 尝试不同的图像生成模式
  • 将图片转换成视频
  • 切换中英文界面和浅色/深色主题
🔧 遇到问题？
  • 页面打不开：检查终端是否显示 "ready"，确认项目已启动
  • 显示空白页：按 Command (⌘) + Shift + R 强制刷新
  • AI 功能不可用：检查 .env.local 文件中的 API 密钥是否正确配置
  • 图片上传失败：检查网络连接，确保能访问 Google 服务

💡 使用技巧

🎨 如何使用图像编辑功能？

  1. 在界面上选择一个编辑模式（比如"背景替换"、"图像扩展"等）
  2. 点击"上传图片"按钮，选择要编辑的图片
  3. 如果需要局部编辑，用画笔圈出想要修改的区域
  4. 点击"生成"按钮
  5. 等待几秒钟，AI 就会生成新的图片

🎬 如何生成视频？

  1. 先生成或上传一张图片
  2. 在图片下方找到"生成视频"按钮
  3. 点击按钮，等待视频生成
  4. 生成完成后可以预览和下载视频

🔄 如何连续编辑？

  1. 生成一张图片后
  2. 点击"发送到新玩法"按钮
  3. 选择下一个编辑模式
  4. 继续编辑，无限创意！

❓ 常见问题解答

Q1: 为什么我的图片生成很慢？

A: 这取决于：

  • 你的网络速度（需要连接到 Google 服务器）
  • 图片的大小和复杂度
  • 当前 API 服务器的负载

通常需要等待 5-30 秒，请耐心等待。

Q2: 项目停止后如何再次启动？

A: 很简单：

  1. 打开终端
  2. 进入项目文件夹：cd /Volumes/ORICO/编程项目/Nano-Bananary
  3. 运行：npm run dev
  4. 在浏览器中访问：http://localhost:5173

Q3: 可以在其他电脑上访问吗？

A: 默认只能在本地访问。如果想让同一网络下的其他设备访问：

  1. 运行：npm run dev -- --host
  2. 终端会显示一个 Network 地址
  3. 其他设备用这个地址访问即可

Q4: 如何更新 API 密钥？

A:

  1. 打开项目文件夹中的 .env.local 文件
  2. 修改 GEMINI_API_KEY= 后面的值
  3. 保存文件
  4. 在终端中按 Control (⌃) + C 停止项目
  5. 重新运行 npm run dev

Q5: 如何部署到线上？

A: 如果想让所有人都能访问（不只是本地），可以部署到 Vercel 或 Netlify 等平台：

  1. 注册 Vercel 账号
  2. 安装 Vercel CLI：npm install -g vercel
  3. 在项目文件夹运行：vercel
  4. 按照提示完成部署
  5. 记得在 Vercel 后台配置环境变量 GEMINI_API_KEY

🛑 如何停止项目

当你不再使用项目时，应该停止它以释放电脑资源：

  1. 回到运行项目的终端窗口
  2. 按下 Control (⌃) + C
  3. 看到终端返回到正常的命令行提示符，说明已停止
  4. 现在可以关闭终端窗口了
⚠️ 注意：

停止项目后，浏览器中的页面将无法继续使用，需要重新启动项目。
